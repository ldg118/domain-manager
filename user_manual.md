## 系统简介

域名管理系统是一个基于 Cloudflare Pages 的轻量级应用，帮助您轻松管理和监控多个域名的状态、到期时间和SSL证书等信息。系统采用简洁直观的界面设计，支持移动端和桌面端访问，让您随时随地管理您的域名资产。

## 功能特点

- **域名管理**：添加、编辑、删除、导入、导出域名信息
- **状态监控**：自动检查域名在线状态
- **到期提醒**：设置域名到期提醒时间
- **多注册商支持**：支持多个域名注册商的信息记录
- **SSL证书管理**：申请、续期和监控SSL证书
- **Telegram 通知**：支持通过 Telegram 发送到期提醒
- **响应式设计**：支持移动端和桌面端访问
- **安全认证**：基于用户名密码的访问控制

## 开始使用

### 登录系统

1. 访问系统部署的网址
2. 输入管理员用户名和密码
3. 点击"登录"按钮

### 界面概览

登录后，您将看到系统的主界面，包括以下几个主要部分：

- **仪表盘**：显示域名和证书的概览信息
- **域名管理**：管理您的域名列表
- **证书管理**：管理SSL证书
- **系统设置**：配置系统参数和通知设置

## 域名管理

### 添加域名

1. 在左侧菜单中点击"域名管理"
2. 点击"添加域名"按钮
3. 填写域名信息：
   - 域名（必填）
   - 注册商
   - 注册商链接
   - 注册日期
   - 到期日期（必填）
   - 服务类型
   - 备注
4. 点击"保存"按钮

### 编辑域名

1. 在域名列表中找到要编辑的域名
2. 点击"编辑"按钮
3. 修改域名信息
4. 点击"保存"按钮

### 删除域名

1. 在域名列表中找到要删除的域名
2. 点击"删除"按钮
3. 在确认对话框中点击"确认"

### 导入域名

系统支持通过JSON文件批量导入域名数据：

1. 点击"导入"按钮
2. 选择JSON文件
3. 点击"上传"按钮

导入格式示例：

```json
[
  {
    "domain": "example.com",
    "registrar": "Cloudflare",
    "registrar_link": "https://dash.cloudflare.com",
    "registrar_date": "2023-01-01",
    "expiry_date": "2024-01-01",
    "service_type": "网站",
    "memo": "主站"
  },
  {
    "domain": "example.org",
    "registrar": "Namecheap",
    "registrar_link": "https://www.namecheap.com",
    "registrar_date": "2023-02-15",
    "expiry_date": "2024-02-15",
    "service_type": "API服务",
    "memo": "API文档站点"
  }
]
```

### 导出域名

1. 点击"导出"按钮
2. 系统将生成JSON文件并自动下载

## SSL证书管理

### 申请证书

1. 在左侧菜单中点击"证书管理"
2. 点击"申请证书"按钮
3. 选择要申请证书的域名
4. 输入通用名称（可选）
5. 点击"申请"按钮

### 查看证书

1. 在证书列表中找到要查看的证书
2. 点击"详情"按钮
3. 查看证书的详细信息

### 续期证书

1. 在证书列表中找到要续期的证书
2. 点击"续期"按钮
3. 在确认对话框中点击"确认"

### 删除证书

1. 在证书列表中找到要删除的证书
2. 点击"删除"按钮
3. 在确认对话框中点击"确认"

## 状态监控

### 检查域名状态

1. 在左侧菜单中点击"状态监控"
2. 点击"检查所有域名"按钮
3. 系统将自动检查所有域名的在线状态
4. 查看检查结果

### 检查证书状态

1. 在左侧菜单中点击"状态监控"
2. 点击"检查所有证书"按钮
3. 系统将自动检查所有证书的有效性
4. 查看检查结果

## 到期提醒

### 配置提醒设置

1. 在左侧菜单中点击"系统设置"
2. 在"通知设置"部分，配置以下参数：
   - Telegram 机器人令牌
   - Telegram 聊天 ID
   - 提前提醒天数
3. 点击"保存"按钮

### 测试通知

1. 在"通知设置"部分，点击"测试通知"按钮
2. 如果配置正确，您将在 Telegram 中收到测试消息

### 手动发送提醒

1. 在左侧菜单中点击"状态监控"
2. 点击"发送域名到期提醒"或"发送证书到期提醒"按钮
3. 系统将向 Telegram 发送提醒消息

## 系统设置

### 修改密码

1. 在左侧菜单中点击"系统设置"
2. 在"账户设置"部分，输入当前密码和新密码
3. 点击"修改密码"按钮

### 配置通知设置

1. 在左侧菜单中点击"系统设置"
2. 在"通知设置"部分，配置 Telegram 相关参数
3. 点击"保存"按钮

## 常见问题

### 如何获取 Telegram 机器人令牌？

1. 在 Telegram 中搜索 @BotFather
2. 发送 /newbot 命令
3. 按照提示创建机器人
4. 获取机器人令牌

### 如何获取 Telegram 聊天 ID？

1. 在 Telegram 中搜索 @userinfobot
2. 发送任意消息
3. 机器人将返回您的聊天 ID

### 域名状态检查频率是多少？

系统不会自动检查域名状态，您需要手动触发检查。建议每天或每周定期检查一次。

### 如何设置自动续期证书？

在证书详情页面，您可以启用或禁用自动续期选项。启用后，系统将在证书到期前自动尝试续期。
